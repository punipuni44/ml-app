## ■ 2. 開発環境構築（Environment Setup）

---

### ◆ 仮想環境の作成

```powershell
py -m venv venv
```

#### ● コマンド分解

| 要素   | 意味                        |
| ---- | ------------------------- |
| py   | Pythonランチャー（Windows用）     |
| -m   | モジュールを実行するオプション（moduleの略） |
| venv | 仮想環境を作るモジュール              |
| venv | フォルダ名（自由に変更可能）            |

---

#### ● 仮想環境とは？

👉 **プロジェクトごとにPythonの環境を分離する仕組み**

例：

* Aプロジェクト → pandas 1.x
* Bプロジェクト → pandas 2.x

👉 競合を防ぐ

---

### ◆ 仮想環境の起動

```powershell
venv\Scripts\activate
```

👉 `(venv)` が表示される = 仮想環境ON

---

### ◆ ライブラリのインストール

```powershell
pip install pandas matplotlib scikit-learn
```

| ライブラリ        | 役割    |
| ------------ | ----- |
| pandas       | データ操作 |
| matplotlib   | グラフ   |
| scikit-learn | 機械学習  |

## ■ 3. Pythonコード完全解説（最重要）

---

### ◆ ① データ作成

```python
data = {
    "day": [1,2,3,4,5],
    "sales": [100,150,200,250,300]
}
```

#### ● 用語：辞書（dictionary）

* キー（key）と値（value）のセット

---

#### ● なぜこの形？

👉 機械学習では「表形式データ」が基本

| day | sales |
| --- | ----- |
| 1   | 100   |

---

---

### ◆ ② DataFrameに変換

```python
df = pd.DataFrame(data)
```

#### ● DataFrameとは？

👉 pandasのデータ構造
👉 Excelの表と同じ概念

---

### ◆ ③ 特徴量と目的変数の分離（超重要）

```python
X = df[["day"]]
y = df["sales"]
```

---

### ◆ 用語解説（ここが最重要）

| 用語   | 英語      | 意味     |
| ---- | ------- | ------ |
| 特徴量  | Feature | 入力データ  |
| 目的変数 | Target  | 予測したい値 |

---

### ◆ なぜ分けるのか？

👉 モデルに「入力 → 出力」の関係を学習させるため

---

### ◆ なぜ df[["day"]]？

```python
df["day"]   # 1次元（Series）
df[["day"]] # 2次元（DataFrame）
```

👉 機械学習モデルは「2次元配列」を要求する

---

### ◆ ④ モデル作成

```python
model = LinearRegression()
```

#### ● 用語：線形回帰（Linear Regression）

👉 データを直線で近似するモデル

---

### ◆ 数式イメージ

```
sales = a × day + b
```

---

### ◆ ⑤ 学習（fit）

```python
model.fit(X, y)
```

#### ● fitとは？

👉 **データからルールを学習する処理**

---

#### ● 内部で何が起きているか

* a（傾き）とb（切片）を計算
* 誤差を最小化

---

### ◆ ⑥ 予測（predict）

```python
pred = model.predict([[6]])
```

---

#### ● なぜ [[6]]？

👉 2次元で渡す必要がある

---

### ◆ ⑦ 可視化

```python
plt.scatter(df["day"], df["sales"])
plt.plot(df["day"], model.predict(X))
plt.savefig("result.png")
```

---

#### ● 用語

| 関数      | 意味   |
| ------- | ---- |
| scatter | 実データ |
| plot    | 予測線  |

---

#### ● なぜ重要？

👉 モデルの正しさを視覚的に確認できる

---
